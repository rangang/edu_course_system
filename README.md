# edu_course_system
单体架构项目-教育课程管理系统

## 后端技术

- Web层：Servlet->前端控制器、Filter->过滤器、BeanUtils->数据封装
- Service层：业务处理
- dao层：Mysql->数据库、Druid->数据库连接池、DBUtils:->操作数据库

## 项目开发环境
- 开发工具：后端->IDEA、数据库->Navicat Premium
- 开发环境：JDK11、Maven 3.6.3、MySQL 5.7
